# Весенний Манифест · Юля

Интерактивный подарок на день рождения — 15 апреля 2026.

## Как запустить локально

Просто откройте `index.html` в браузере.

## Как разместить на GitHub Pages

1. Создайте новый репозиторий на GitHub (например `manifest`)
2. Загрузите все файлы из этой папки
3. Перейдите в Settings → Pages → Source: `main` branch, `/root`
4. Сайт будет доступен по адресу: `https://ВАШ_НИК.github.io/manifest`

## Что нужно заменить перед запуском

В файле `index.html` найдите и замените:

- **Слова друзей** (блок «Суперсилы»): ищите `var FRIENDS=` — замените имена и слова
- **Послания от Лены** (блок «Муми-долина»): ищите `var SPOTS=` — замените тексты `m:`
- **Текст подарка** (финальный блок): ищите `id="mgift"` — замените текст внутри

## Технические детали

- Один HTML файл, все картины встроены (base64)
- Не требует сервера, интернет нужен только для Google Fonts
- Мобильная адаптация включена
- Письмо через год отправляется через FutureMe.org
